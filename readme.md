# html5j-trans

2015/1/25のHTML5 conference で、英語 => 日本語翻訳Tweet表示用に作ったWebアプリ

## how to install
```
$ npm install
$ cd conf
$ cp twitter.conf.smpl twitter.conf
// edit twitter.conf
```
